# NexaCRM – Feature Enhancement Tasks

## Customer Module
- Added Industry field to Add/Edit Customer.
- Industry is stored in the database and displayed throughout the application.

## Dashboard
- Dashboard Quick Action buttons now use the same forms, validations, APIs, and business logic as the sidebar.
- Removed:
  - Team Performance
  - Team Score
  - Online Score

## User Module
- Removed the Score field.

## Campaign Module
- Added searchable Customer selection.
- Campaigns are linked to existing customers.
- Customer name displayed in Campaign List and Details.
- Handles deleted customers gracefully.
- Removed:
  - Budget
  - Expected Income
  - Actual Income
  - ROI
- Campaign statistics use different colors for each status.

## Lead Notifications
- Employees receive notifications whenever a lead is assigned or reassigned.

## Communications
### Updated Table Columns
- Channel
- Receiver Name
  - Lead/Customer shown below the name

- Sent By
- Date

- Replaced browser alert/confirm dialogs with a modern popover confirmation.

## Reports
- Print now downloads reports in a properly formatted printable file.

## General Improvements
- Improved consistency across modules.
- Unified forms and validations.
- Better UI/UX.
- Dynamic data synchronization across the CRM.

## Dashboard – Lead Funnel
- Replace the current Lead Funnel, which only displays:
  - New
  - Qualified
  - Proposal
  - Won
- The Lead Funnel must dynamically display **all lead stages** using real CRM data:
  - New
  - Contacted
  - Interested
  - Qualified
  - Proposal Sent
  - Negotiation
  - Converted
  - Lost
- Each stage should show the live count of leads currently in that stage.
- Remove hardcoded/static values and update the funnel automatically whenever a lead is created, updated, converted, deleted, or its status changes.
- Use a professional funnel chart (or horizontal funnel visualization) that accurately represents the lead lifecycle instead of the current simplified graph.
